# JIT Defect Risk from Commit Messages
Fine-tune DistilBERT on commit messages to produce commit-time defect-risk signals (Java/.NET).

## Folders
data/raw (per-repo CSVs) • data/processed (merged/splits) • src (scripts) • notebooks (exploration)
reports (logs/analysis) • configs (experiment settings) • docs (protocols/cards) • artifacts (models)
